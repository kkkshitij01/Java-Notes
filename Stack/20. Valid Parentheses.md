Given a string `s` containing just the characters `'('`, `')'`, `'{'`, `'}'`, `'['` and `']'`, determine if the input string is valid.

An input string is valid if:

1. Open brackets must be closed by the same type of brackets.
2. Open brackets must be closed in the correct order.
3. Every close bracket has a corresponding open bracket of the same type.

**Example 1:**

**Input:** s = "()"

**Output:** true

**Example 2:**

**Input:** s = `"()[]{}"`

**Output:** true

**Example 3:**

**Input:** s = "(]"

**Output:** false

**Example 4:**

**Input:** s = `"([])"`

**Output:** true

**Example 5:**

**Input:** s = `"([)]"`

**Output:** false

---

# Optimal Solution : 
### ⚙️ **Function:** `isValid(String s)`

• **Goal:**  
 → Check if a string containing `'('`, `')'`, `'{'`, `'}'`, `'['`, `']'` has **valid parentheses** — meaning every opening bracket has a correctly ordered closing one.
 
• **Step 1:**  
 → Create a `Stack<Character>` to keep track of **opening brackets**.
 
• **Step 2:**  
 → Traverse the string character by character.
 → **If the character is an opening bracket** (`'('`, `'{'`, `'['`):  
  – Push it onto the stack.
 → **If the character is a closing bracket** (`')'`, `']'`, `'}'`):  
  – If the stack is empty → return `false` (no opening bracket to match).  
  – Else check if the **top of the stack** has the matching opening bracket:  
   • `'('` for `')'`  
   • `'{'` for `'}'`  
   • `'['` for `']'`  
  – If it matches → pop the top bracket (pair matched).  
  – If not → return `false` (mismatch found).

• **Step 3:**  
 → After the loop, check if the stack is empty.  
  – If empty → all brackets were matched → return `true`.  
  – If not empty → there are unmatched opening brackets → return `false`.
``` java
class Solution {

    public boolean isValid(String s) {

        Stack<Character> stack = new Stack<>(); // stack to store opening brackets

        // Loop through every character in the string
        for (int i = 0; i < s.length(); i++) {

            char ch = s.charAt(i);

            // If current character is an opening bracket, push it to the stack
            if (ch == '[' || ch == '(' || ch == '{') {
                stack.push(ch);
            } 
            // If it's a closing bracket
            else {
                // If stack is empty, there's no opening bracket to match
                if (stack.isEmpty()) {
                    return false;
                }

                // Check if the top of the stack matches the type of closing bracket
                if (ch == ')' && stack.peek() == '(' ||
                    ch == ']' && stack.peek() == '[' ||
                    ch == '}' && stack.peek() == '{') {

                    // If matched, remove the top element from stack
                    stack.pop();
                } 
                // If it doesn’t match, parentheses are invalid
                else {
                    return false;
                }
            }
        }

        // If stack is empty, all brackets matched correctly
        // Otherwise, there are unmatched opening brackets
        return stack.isEmpty();
    }
}
```
