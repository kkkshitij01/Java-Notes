Given an input string `s`, reverse the order of the **words**.

A **word** is defined as a sequence of non-space characters. The **words** in `s` will be separated by at least one space.

Return _a string of the words in reverse order concatenated by a single space._

**Note** that `s` may contain leading or trailing spaces or multiple spaces between two words. The returned string should only have a single space separating the words. Do not include any extra spaces.

**Example 1:**

**Input:** s = "the sky is blue"
**Output:** "blue is sky the"

**Example 2:**

**Input:** s = "  hello world  "
**Output:** "world hello"
**Explanation:** Your reversed string should not contain leading or trailing spaces.

**Example 3:**

**Input:** s = "a good   example"
**Output:** "example good a"
**Explanation:** You need to reduce multiple spaces between two words to a single space in the reversed string.

---

# BruteForce 

• **Step 1:**  
 → Use `s.split(" +")` to split the string into an array of words.  
 → `" +"` (regex) ensures multiple spaces are treated as a single separator.
 
• **Step 2:**  
 → Create a `StringBuilder sb` to build the reversed sentence efficiently.

• **Step 3:**  
 → Loop through the array of words **from the end to the beginning**.  
 → For each word:  
  – Append the word to `sb`.  
  – Append a space `" "` after each word.

• **Step 4:**  
 → After the loop, use `sb.toString().trim()` to remove the trailing space at the end.

• **Step 5:**  
 → Return the final reversed string.


```java
class Solution {
    public String reverseWords(String s) {
        // Split the string by one or more spaces (" +")
        // This automatically removes extra spaces between words
        String ans[] = s.split(" +");

        StringBuilder sb = new StringBuilder();

        // Loop backward through the array to reverse the order of words
        for (int i = ans.length - 1; i >= 0; i--) {
            sb.append(ans[i]); // add the word
            sb.append(" ");    // add a space after each word
        }

        // Convert StringBuilder to string and trim the extra space added after the last word
        return sb.toString().trim();
    }
}


```

# Optimal Solution : 

• **Step 1:**  
 → Start from the end of the string (`i = s.length() - 1`).  
 → Create a `StringBuilder sb` to build the final reversed string efficiently.

• **Step 2:**  
 → Use a loop to process each word in reverse order until the beginning of the string.  
 → **Skip trailing spaces:**  
  – While `i >= 0` and the current character is a space → move `i--`.  
  – If `i < 0`, break the loop (string fully processed).

• **Step 3:**  
 → Set `j = i` to mark the end of the current word.  
 → Move `i` left until a space or start of string is found → this identifies the word boundaries.

• **Step 4:**  
 → Append the current word from `s[i+1]` to `s[j]` into the `StringBuilder`.  
 → Add a space after the word for separation.

• **Step 6:**  
 → Once all words are processed, return `sb.toString().trim()` to remove any trailing space.

``` java
class Solution {
    public String reverseWords(String s) {
        int i = s.length() - 1;          // Start from the end of the string
        StringBuilder sb = new StringBuilder(); // To store the reversed result
        
        while (i >= 0) {
            // Step 1: Skip any trailing spaces
            while (i >= 0 && s.charAt(i) == ' ') i--;
            if (i < 0) break;             // End of string reached
            
            int j = i;
            // Step 2: Move i to the beginning of the current word
            while (i >= 0 && s.charAt(i) != ' ') i--;
            
            // Step 3: Append the current word (from i+1 to j)
            sb.append(s, i + 1, j + 1).append(' ');
            
        }
        
        // Step 5: Trim any trailing space and return the result
        return sb.toString().trim();
    }
}
```
