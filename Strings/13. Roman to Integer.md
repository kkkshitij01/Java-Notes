Roman numerals are represented by seven different symbols: `I`, `V`, `X`, `L`, `C`, `D` and `M`.

**Symbol**       **Value**
I             1
V             5
X             10
L             50
C             100
D             500
M             1000

For example, `2` is written as `II` in Roman numeral, just two ones added together. `12` is written as `XII`, which is simply `X + II`. The number `27` is written as `XXVII`, which is `XX + V + II`.

Roman numerals are usually written largest to smallest from left to right. However, the numeral for four is not `IIII`. Instead, the number four is written as `IV`. Because the one is before the five we subtract it making four. The same principle applies to the number nine, which is written as `IX`. There are six instances where subtraction is used:

- `I` can be placed before `V` (5) and `X` (10) to make 4 and 9. 
- `X` can be placed before `L` (50) and `C` (100) to make 40 and 90. 
- `C` can be placed before `D` (500) and `M` (1000) to make 400 and 900.

Given a roman numeral, convert it to an integer.


# Approach :

### **Function:** `romanToInt(String s)`

• **Step 1:**  
 → Create a `HashMap<Character, Integer>` that stores the **Roman symbols** and their corresponding **values**:  
  `I → 1`, `V → 5`, `X → 10`, `L → 50`, `C → 100`, `D → 500`, `M → 1000`.

• **Step 2:**  
 → Initialize `ans = 0` to store the result.  
 → Loop through the string from index `0` to `len - 2`.

• **Step 3:**  
 → For each character, get its value: `val = map.get(s.charAt(i))`.  
 → Compare it with the **next** character’s value:  
  – If `val < next value`, it means we have a **subtractive pair** (like `IV = 4`, `IX = 9`) → subtract `val` from `ans`.  
  – Else, add `val` to `ans`.

• **Step 4:**  
 → After the loop, add the value of the **last Roman symbol** (`map.get(s.charAt(len - 1))`) to the result.

• **Step 5:**  
 → Return `ans` as the final integer value.
```java
class Solution {
    public int romanToInt(String s) {
        // Step 1: Create a map for Roman numerals and their integer values
        HashMap<Character, Integer> map = new HashMap<>();
        map.put('I', 1);
        map.put('V', 5);
        map.put('X', 10);
        map.put('L', 50);
        map.put('C', 100);
        map.put('D', 500);
        map.put('M', 1000);

        int len = s.length();
        int ans = 0;

        // Step 2: Loop through all characters except the last one
        for (int i = 0; i < len - 1; i++) {
            int val = map.get(s.charAt(i)); // current Roman character value

            // Step 3: If current value is smaller than the next one,
            // it means we need to subtract (like IV = 4)
            if (val < map.get(s.charAt(i + 1))) {
                ans -= val;
            } 
            // Otherwise, add the value normally
            else {
                ans += val;
            }
        }

        // Step 4: Add the value of the last character (always added)
        return ans + map.get(s.charAt(len - 1));
    }
}

```
