
Given two strings `s` and `t`, return `true` if `t` is an anagram of `s`, and `false` otherwise.

**Example 1:**

**Input:** s = "anagram", t = "nagaram"

**Output:** true

**Example 2:**

**Input:** s = "rat", t = "car"

**Output:** false

--- 

# Approach

### STEP 1

```
if (s.length() != t.length()) return false;
```

- **What:** Immediate length comparison.
    
- **Why:** Different lengths cannot be rearrangements.
    
- **Optimization:** O(1) check saves processing time.
    

---

### STEP 2

```
char count[] = new char[26];
```

- **What:** Frequency bucket for 'a' to 'z'.
    x- **Mapping:** Index 0=′a′, Index 25=′z′.
    
- **Space:** Constant O(1) (fixed size).
    

---

### STEP 3

```
for (int i = 0; i < s.length(); i++) count[s.charAt(i) - 'a']++;
```

- **What:** Build frequency profile for s.
    
- **Logic:** Convert char to index (`'c'-'a'=2`) and **Increment (+)**.
    
- **Visual:** `s="a"` → `Index 0: +1`.
    

---

### STEP 4

```
for (int i = 0; i < s.length(); i++) count[t.charAt(i) - 'a']--;
```

- **What:** Cancel out frequencies using t.
    
- **Logic:** Convert char to index and **Decrement (-)**.
    
- **Visual:** `t="a"` → `Index 0: -1`. Result is `0`.
    

---
### PART 2


#### THE VALIDATION PHASE

### STEP 5

```
for (int i : count) if (i != 0) return false;
return true;
```

- **What:** Check if array is empty (all zeros).
    
- **Logic:**
    
    - **> 0**: s has extra chars.
        
    - **< 0**: t has extra chars.
        
    - **0**: Perfect Match.


```java
class Solution {
    /**
     * Checks if a string 't' is an anagram of a string 's'.
     * An anagram means 't' is created by rearranging the letters of 's'.
     */
    public boolean isAnagram(String s, String t) {
        
        // --- Step 1: Basic Length Check ---
        
        // If the two strings don't have the exact same number of letters, 
        // they can't possibly be anagrams. So, we immediately return 'false'.
        if (s.length() != t.length()) {
            return false;
        }

        // --- Step 2: Set up a Frequency Counter ---
        
        // We create an array of size 26. This array will be used to count 
        // how many times each lowercase English letter (a-z) appears.
        // Index 0 represents 'a', Index 1 represents 'b', and so on up to Index 25 for 'z'.
        int count[] = new int[26];
        
        // --- Step 3: Count Characters in the First String (s) ---
        
        // We loop through every character in the first string, 's'.
        for (int i = 0; i < s.length(); i++) {
            // s.charAt(i) gives us the current letter (e.g., 'c').
            // By subtracting 'a' (e.g., 'c' - 'a'), we get its position (index 2).
            // We use this index to increase the count for that specific letter.
            count[s.charAt(i) - 'a']++; 
        }
        
        // --- Step 4: Subtract Characters from the Second String (t) ---

        // Now, we loop through every character in the second string, 't'.
        for (int i = 0; i < s.length(); i++) {
            // We use the same method to find the letter's index.
            // But this time, we DECREASE the count for that letter.
            // If 't' is a true anagram, this count should go down to 0 for every letter.
            count[t.charAt(i) - 'a']--;
        }
        
        // --- Step 5: Final Check ---
        
        // We check every single position (a to z) in our 'count' array.
        for (int i = 0; i < count.length; i++) {
            // If any spot in the array is NOT 0, it means that letter appeared 
            // a different number of times in 's' versus 't'.
            // For example, if count[2] is 1, it means 'c' appeared one more time in 's' than in 't'.
            if (count[i] != 0) {
                // Since the counts don't match up, they are not anagrams.
                return false;
            }
        }
        
        // If we make it through the entire loop and every count is 0, 
        // it means every letter appeared the exact same number of times in both strings.
        // Therefore, they ARE anagrams!
        return true;
    }
}
```
