### LEET CODE : 1392 : LONGEST HAPPY PREFIX

A string is called a **happy prefix** if is a **non-empty** prefix which is also a suffix (excluding itself).

Given a string `s`, return _the **longest happy prefix** of_ `s`. Return an empty string `""` if no such prefix exists.

**Example 1:**

**Input:** s = "level"
**Output:** "l"
**Explanation:** s contains 4 prefix excluding itself ("l", "le", "lev", "leve"), and suffix ("l", "el", "vel", "evel"). The largest prefix which is also suffix is given by "l".

**Example 2:**

**Input:** s = "ababab"
**Output:** "abab"
**Explanation:** "abab" is the largest prefix which is also suffix. They can overlap in the original string.

# BRUTE FORCE


```java
class Solution {
    public String longestPrefix(String s) {
        
        int n = s.length();
        
        
        // We start checking from the LONGEST possible prefix/suffix length (n-1) 
        // down to the smallest (length 1).
        // If we find a match, it's guaranteed to be the longest one, so we stop immediately.
        for (int i = n - 1; i >= 1; i--) {
            
            // 'i' represents the current length we are checking.
            
            // --- Step 2: Extract Prefix and Suffix ---

            // Get the prefix of the string with the current length 'i'.
            // s.substring(0, i) takes characters from the start (index 0) up to index 'i'.
            String prefix = s.substring(0, i);
            
            // Get the suffix of the string with the current length 'i'.
            // s.substring(n-i) takes characters from (n-i) until the end of the string.
            String sufix = s.substring(n - i);
            
            // --- Step 3: Compare ---

            // Check if the extracted prefix and the extracted suffix are exactly the same.
            if (prefix.equals(sufix)) {
                
                // If they match, we've found the longest possible one 
                // (because we started checking from the longest length).
                // We return this matching prefix string and exit the function.
                return s.substring(0, i);
            }
        }
        
        // --- Step 4: No Match Found ---
        
        // If the loop finishes without finding any match (meaning no prefix of length 1, 2, 3... 
        // matched the corresponding suffix), we return an empty string.
        return "";
    }
}
```
